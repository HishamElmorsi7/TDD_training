## Want to play the game ?

### Here are the steps:

**(1)** clone the project
**(2)** upload tower_of_hanoi.rb file to pry
```ruby
load "spec/tower_of_hanoi.rb"
```
**(3)** create an object of the game 
```ruby
game = TowerOfHanoi.new
```
**(4)** play the game
```ruby
game.play
```
